# Tmux Configuration

Personal tmux configuration with sensible defaults and useful plugins.

## Installation

1. Clone the repository:
```bash
git clone https://github.com/y37y/tmux.git ~/.config/tmux
```

2. Create symlink:
```bash
ln -s ~/.config/tmux/.tmux.conf ~/.tmux.conf
```

3. Install plugins:
Press `prefix` + `I` (capital i) to install the plugins
